Eu, Daniel Mafra, vou ajudar você da seguinte forma:

Criação de site Single Page

É um site de uma única página dividida em várias seções.

Através do menu o usuário pode navegar entre as seções que a página possui.

Criação de site Multi Page

É um site onde cada uma das seções possui uma página específica.

Através do menu o usuário pode navegar entre cada uma das páginas que o site possui.

Criação de Bot Page

É uma página interativa no modelo de um chatbot.

O usuário recebe perguntas no formato de mensagens e seleciona a opção de resposta que faz sentido pra ele.

Começamos com um briefing e terminamos com um belo projeto...
